# Annotation Review Dashboard

A responsive JavaScript web application that simulates an AI data annotation and quality review workflow.

The dashboard allows reviewers to examine text samples, assign sentiment labels, record confidence levels, flag ambiguous cases and monitor annotation progress.

## Features

- Review structured text samples
- Assign Positive, Neutral, or Negative sentiment labels
- Record annotation confidence from 1 to 5
- Flag ambiguous or difficult samples
- Add reviewer notes
- Track completed and remaining samples
- View annotation statistics
- Review recent annotation history
- Save progress with browser local storage
- Clear saved annotation history
- Responsive design for desktop and mobile devices

##Annotation Workflow
1. Read the displayed text sample.
2. Select a sentiment label.
3. Choose a confidence level.
4. Flag the sample when it is ambiguous.
5. Add optional reviewer notes.
6. Submit the annotation.
7. Review the updated statistics and annotation history.
